## ✨ Overview

**Freut Mich** is a modern full-stack e-commerce shopping application built for speed, performance, and clean user experience.  
It allows users to browse products, manage carts, and enjoy a smooth shopping journey.

---

## 🚀 Features

- 🔐 Secure Login / Signup (JWT Authentication)
- 🛒 Add to Cart / Remove from Cart
- 📦 Product Listing with Categories
- ❤️ Wishlist Feature
- 📱 Fully Responsive Design
- ⚡ Fast API integration
- 🎯 Clean UI with modern design
- 🧾 Order management (if implemented)

## 🧑‍💻 Tech Stack
